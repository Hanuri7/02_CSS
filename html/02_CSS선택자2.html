기본 속성 선택자

    - 요소에 작성된 속성을 이용해 선택하는 선택자
    - 속성, 속성 값이 일치하는 요소만을 선택

    [작성법]
    
      1. 같은 속성을 가진 요소 선택
        [속성명]{CSS 코드;}

      2. 같은 속성, 속성 값을 가진 요소 선택
        [속성명=속성값]{CSS 코드;}
        //속성 값은 '',"" 사용 가능

동위 선택자(+, ~)

    동위(동등한 위치 == 형제 관계)
    다음(뒤, 아래)에 위치한 요소를 선택하는 선택자

    1. A 바로 다음에 위치한 요소를 하나 선택(+)
    
      A선택자 + B선택자 {CSS 코드;}
    
    2. A 바로 다음에 위치한 모든 B요소를 하나 선택(~, Tilde(틸드))
    
      A선택자 ~ B선택자 {CSS 코드;}
