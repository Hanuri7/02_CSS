CSS(Cascading Style Sheets)

    Cascading : 종속, 여러 단으로 된 폭포

    * 마크업 언어(html)가 실제 화면에 표시되는 방법을 기술한 언어
    * 하나의 요소에 여러 CSS 코드를 겹쳐서 원하는 스타일 표시
  
    [용어 정리]
    id(식별자) 속성
    - 하나의 html 파일 내에 단 하나만 존재할 수 있음(중복 X)
      -> 요소를 구분하는 속성
    
    - id를 나타내는 기호 : #

    class(학급, 모임) 속성
    - 요소 하나가 여러 class에 존재 가능
      -> 같은 class를 가지는 요소가 존재 가능

    sodptj wjrdydgkrh tlvdms tmxkdlf
    html 파일

태그 선택자

    - 같은 태그를 모두 찾아서 선택
  • test1
  • test2
  • test3
  • test4
  • test5

ID(#) 선택자

    - html 문서내에 같은 id를 가지는 요소를 선택

    [작성법]
    #id속성값 {css 코드;}
  • test1
  • test2
  • test3

class(.) 선택자

    - html 문서내에 같은 class를 가지는 요소를 선택

    [작성법]
    .class속성값 {css 코드;}

    1. 단독 사용

      -> 선택된 요소의 모든 자식/형재/후손 가능
